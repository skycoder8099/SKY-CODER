Introduction to Computer

A computer is a machine that can be programmed to automatically carry out sequences of arithmetic or logical operations (computation). Modern digital electronic computers can perform generic sets of operations known as programs. These programs enable computers to perform a wide range of tasks. The term computer system may refer to a nominally complete computer that includes the hardware, operating system, software, and peripheral equipment needed and used for full operation; or to a group of computers that are linked and function together, such as a computer network or computer cluster.

What is Computer

A computer is an electronic device that is capable of accepting, storing and proccessing data according to instruction to performed various operations. Every computer OS a combination of two basic components: Software and Hardware. Computer can be divided in three main catagories:
(a) Analog Computer. (b) Digital Computer. (c) Hybrid Computer.

Hardware and Software

Computer Hardware: Computer Hardware refers to the physical components of a computer system.
Computer Software: Software is a set of instructions that tells the computer about the tasks to be performed and how these tasks are to be performed.

About input and output devices

I/O stands for input/output, which is the process by which computers communicate with the outside world. I/O devices are hardware that allows a computer to send and receive data from other systems or peripherals devices.

    Here are some things to know about I/O devices:
    1. Input devices are used to enter data into the computer.
    2. Output devices are used to display data from the computer.
Input and Output
Input refers to data sent to a computer, while output refers to data that has been processed and comes from the computer.
Examples
Some examples of input devices include keyboards and mice, while monitors and printers are examples of output devices.

About CPU

The CPU, or central processing unit, is the brain of the computer. It is also known as the processor or microprocessor, is the primary component of a computer that executes most instructions and tasks.
The CPU is the brain of a computer, performing mathematical operations on data and converting it into digital signals. It fetches instructions from memory, executes them, and sends the results back to memory.
The CPU works through a fetch-execute cycle,where it first fetches an instruction from memory,then decodes and executes it.

What is Memory

Computer memory is just like a human brain, which store data and information program for temporeraly or permanently.

    There are two types of memory
    a) Primary memory
    b) Secondary memory
Primary Memory

Primary memory is also known as the main memory of the computer which can store data and information temporarily. It is the memory that is directly accessible by the CPU.


Types of Primary Memory

There are some types of primary memory:

    a) RAM (Random Access Memory)
    b) ROM (Read Only Memory)
    c) EPROM (Erasable Programmable Read Only Memory)
    d) EEPROM (Electrically Erasable Programmable Read Only Memory)

Secondary Memory

Secondary memory is also known as external memory that can store data and information permanently. It is non-volatile memory means stored data and information is not lost even when the power is off.

Types of Secondary Memory

There are some types of secondary memory:

    a) Hard Disk
    b) Floppy Disk
    c) CD/DVD
    d) USB

Internal Memory
Internal memory is also known as main memory or primary memory, which van directly access by the CPU.
External Memory
External memory is also known as secondary memory, It is a storage capacity wich can store data and information permanently.

What is RAM?
RAM is a type of primary memory that stores data and information temporarily. It is used to store the data and information that is being processed by the CPU.

What is ROM?
ROM is a type of primary memory that stores data and information permanently. It is used to store the data and information that is being processed by the CPU.
